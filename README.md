SayGames에서 만든 Rocky Climb 을 모작으로 개발했다. 
플레이어가 여러가지의 산을 올라 정복하는 게임이다. 
돌이 있는 곳에서 화면을 클릭하면 플레이어가 돌을 잡고 산을 등반한다.
단순하고 중독성 있는 플레이 방식이다.
